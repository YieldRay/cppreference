在标头 <locale> 定义
template< class CharT >
bool isalnum( CharT ch, const locale& loc );

检查给定字符按给定本地环境的 std::ctype 刻面是否分类为字母数字字符。

参数

ch - 字符
loc - 本地环境

返回值

若字符被分类为字母数字则返回 true，否则返回 false。

可能的实现

template<class CharT>
bool isalnum(CharT ch, const std::locale& loc)
{
    return std::use_facet<std::ctype<CharT>>(loc).is(std::ctype_base::alnum, ch);
}

示例

演示以不同本地环境使用 isalnum()（OS 特定）

运行此代码
#include <iostream>
#include <locale>
 
int main()
{
    const wchar_t c = L'\u2135'; // 数学符号 aleph
 
    std::locale loc1("C");
    std::cout << "isalnum('ℵ', C locale) 返回 "
               << std::boolalpha << std::isalnum(c, loc1) << '\n';
 
    std::locale loc2("en_US.UTF-8");
    std::cout << "isalnum('ℵ', Unicode locale) 返回 "
              << std::boolalpha << std::isalnum(c, loc2) << '\n';
}

可能的输出： 

isalnum('ℵ', C 本地环境) 返回 false
isalnum('ℵ', Unicode 本地环境) 返回 true
