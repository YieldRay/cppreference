在标头 <experimental/ranges/algorithm> 定义
template< ForwardIterator I1, Sentinel<I1> S1, ForwardIterator I2, Sentinel<I2> S2,

          class Proj = ranges::identity,
          IndirectRelation<I2, projected<I1, Proj>> Pred = ranges::equal_to<> >
I1 find_end( I1 first1, S1 last1, I2 first2, S2 last2,

             Pred pred = Pred{}, Proj proj = Proj{} );
 (1) (范围 TS)
template< ForwardRange R1, ForwardRange R2, class Proj = ranges::identity,

          IndirectRelation<ranges::iterator_t<R2>,
                           projected<ranges::iterator_t<R1>, Proj>> Pred = ranges::equal_to<> >
ranges::safe_iterator_t<R1> find_end( Rng1&& rng1, Rng2&& rng2,

                                      Pred pred = Pred{}, Proj proj = Proj{} );
 (2) (范围 TS)
1) 搜索范围 [first1, last1)（以 proj 投射后）最后一次出现的序列 [first2, last2)。
2) 与 (1) 相同，但用 r1 为第一源范围并用 r2 为第二源范围，如同以 ranges::begin(r1) 为 first1，以 ranges::end(r1) 为 last1，以 ranges::begin(r2) 为 first2，并以 ranges::end(r2) 为 last2。

尽管声明描述如上，算法声明的模板形参的实际数量和顺序是未指定的。从而若在调用算法时使用显式模板实参，则程序很可能不可移植。

参数

first1, last1 - 要检验的元素范围
r1 - 要检验的元素范围
first2, last2 - 要搜索的元素范围
r2 - 要搜索的元素范围
pred - 用于比较元素的谓词
proj - 运用于第一范围中元素的投射

返回值

指向范围 [first1, last1)（经过 proj 投射后）最后出现的序列 [first2, last2) 的开头的迭代器。

如果 [first2, last2) 为空或如果未找到这种序列，则返回与 last1 比较相等的迭代器。

复杂度

最多 S * (N - S + 1) 次运用谓词和投射，其中 S = last2 - first2 且 N = last1 - first1。

注解

仅对范围 [first1, last1) 运用投射。

可能的实现

template<ForwardIterator I1, Sentinel<I1> S1, ForwardIterator I2, Sentinel<I2> S2,
         class Proj = ranges::identity,
         IndirectRelation<I2, projected<I1, Proj>> Pred = ranges::equal_to<>>
I1 find_end(I1 first1, S1 last1, I2 first2, S2 last2,
            Pred pred = Pred{}, Proj proj = Proj{})
{
    I1 result = ranges::next(first1, last1);
    if (first2 == last2)
        return result;
    while (true)
    {
        I1 new_result = ranges::search(first1, last1, first2, last2, pred, proj);
        if (new_result == last1)
            break;
        else
        {
            result = new_result;
            first1 = result;
            ++first1;
        }
    }
    return result;
}
