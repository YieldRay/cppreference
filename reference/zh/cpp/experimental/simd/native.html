在标头 <experimental/simd> 定义
template < class T >
using native = /* 由实现定义 */;
 (并行 TS v2)

native<T> 是 ABI 标签的实现定义别名。这是用于高效的显式向量化的主要标签。

注解

意图是用该 ABI 标签产生对于元素类型 T 最高效的，在当前目标系统上受支持的数据并行执行。对于无 ISA 扩展的目标架构，native<T> 和 compatible<T> 别名很可能相同。对于有 ISA 扩展的结构，编译器标志可能影响 native<T> 别名，而 compatible<T> 会不依赖这种标志而保持相同。
