要求所有值均为唯一值，但 EOPNOTSUPP 与 ENOTSUP 的值可以相同，且 EAGAIN 与 EWOULDBLOCK 的值可以相同。

示例

运行此代码
#include <cerrno>
#include <cstring>
#include <iomanip>
#include <iostream>
 
#define SHOW(x) std::cout << std::setw(15) << #x << ": " << std::strerror(x) << '\n'
 
int main()
{
    std::cout << "Known error codes/messages:\n\n";
 
    SHOW( E2BIG );
    SHOW( EACCES );
    SHOW( EADDRINUSE );
